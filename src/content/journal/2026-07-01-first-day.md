## What I built

Nothing shippable yet — and that's fine. Day one was about **understanding the terrain**.

I set up my local environment:

```bash
node -v   # v24.x
git --version
docker compose up
```

Then I traced a single request from the API gateway all the way down to the model server and sketched the path:

```
Client → API Gateway → Inference Service → Model Server
                    ↘ Vector DB (retrieval)
```

## What I learned

- The AI inference layer lived deeper than I expected, behind two internal services.
- A quick diagram is worth hours of reading source files.

## Difficulties

Finding *where* the LLM call actually happened took a while — the abstraction layers hid it well.
