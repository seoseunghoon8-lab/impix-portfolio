## What I built

This website — a portfolio and journal to record my AI internship at IMPIX. It's built with Next.js, React, TypeScript, and Tailwind CSS.

## What I learned

- **Git basics** — a commit is a save point, and the flow is `add → commit → push`:

```bash
git add -A          # stage what changed
git commit -m "..." # save a snapshot
git push            # send it to GitHub
```

- **GitHub** is an online home for my code that also feeds automatic deployment.
- Installed Node.js, Git, and the GitHub CLI for the very first time.

## Difficulties

I'd never used the terminal for this. A few things tripped me up — PowerShell needs `&` before a quoted program path, and the `!` shortcut only works inside the chat box, not a real terminal.
