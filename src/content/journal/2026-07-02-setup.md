## What I did

- Built this portfolio + journal site with Next.js, React, TypeScript, and Tailwind CSS.
- Installed Node.js, Git, and the GitHub CLI for the first time.
- Made my first commit and pushed the code to GitHub.
- Deployed the site live on Vercel — every push now updates it automatically.
- Set up an easy workflow so I can publish a daily entry just by telling Claude what I did.

## What I learned

- **Git basics** — a commit is a save point, and the flow is `add → commit → push`:

```bash
git add -A          # stage what changed
git commit -m "..." # save a snapshot
git push            # send it to GitHub
```

- GitHub hosts my code and feeds Vercel's automatic deployments.
- Continuous deployment: push to `main` → the live site rebuilds on its own.

## Difficulties

First time really using the terminal — small syntax things tripped me up, like PowerShell needing `&` before a quoted program path.
